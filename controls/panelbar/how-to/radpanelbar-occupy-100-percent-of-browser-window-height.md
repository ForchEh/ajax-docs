# RadPanelBar Occupy 100% of Browser Window Height



## 

* Add the following style in the head tag of your ASPX:

````ASPNET
		 
<style type="text/css">
	html, body, form
	{    
		margin:0px;    
		padding:0px;    
		height:100%;
	}
</style>
				
````



* Set the ExpandMode property of the panelbar control to FullExpandedItem;

* Set Height="100%" in the definition of the panelbar control
